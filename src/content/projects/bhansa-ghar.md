Built a full-stack web application simulating a multi-user food delivery service.

## Key Features

- **Full-Stack Logic**: Developed end-to-end backend functionality with Node.js and Express.
- **NoSQL Schema**: Architected a flexibile MongoDB database for orders, users, and menus.
- **Containerization**: Deployed using Docker, demonstrating infrastructure-as-code and portability.
- **Multi-user Simulation**: Includes complex workflows for customers and delivery tracking.
