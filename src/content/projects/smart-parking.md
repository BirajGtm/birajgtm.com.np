Engineered a two-stage computer vision pipeline to detect parking spaces in images and classify their occupancy status.

## Key Features

- **Computer Vision Pipeline**: Two-stage detection and classification system.
- **Deep Learning**: Trained and deployed a custom Convolutional Neural Network (CNN) in PyTorch/TensorFlow.
- **Accuracy**: Achieved high classification accuracy for real-world parking scenarios.
- **Web Interface**: Built and deployed a user-friendly interface with Gradio for real-time model interaction.
